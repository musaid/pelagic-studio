Use gitmoji prefix format: `:sparkles: <commit message>` (e.g. ✨ Add vision algorithm)

No Co-Authored-By lines — do not add them.

**Why:** User's explicit preference for clean commit history with emoji prefix style.

**How to apply:** Every commit in this repo follows this format. Never add co-authoring footers.
