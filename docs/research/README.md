## How to Read These Documents

Each species document follows the same structure:
1. **Summary** — species name, vision type, key wavelengths
2. **Evidence** — what data exists, what's measured vs. inferred
3. **Photoreceptor Data** — cone types, lambda-max values, weights
4. **Ecological Context** — habitat, hunting behavior, visual ecology
5. **Citations** — full references with DOIs/links where available
6. **Gaps** — what is unknown, what future research could clarify
